# rc_robot_firmware

## Remote-Controlled Robot Firmware

### Requirements

- remote controlled robot
- streams camera video over Wifi
- Programmed in C
- Robot must be
- map keys to driving direction
- Has to be on PCB
- H-bridge from discrete components
- Custom neatly fitted chassis

### Design

- ESP 32 microcontroller
- 4 motors (AWD)
